# Diksha Ai Labs Website

Modern, minimal AI integration website with 3D anti-gravity effects.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Vanilla CSS
- **3D/Animation**: Three.js, React Three Fiber, Rapier (Physics)

## Setup & Running Locally

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```
   This will start the local server, usually at `http://localhost:5173`.

3. **Build for Production**
   ```bash
   npm run build
   ```
